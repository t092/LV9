## 預計修改內容與關卡設計清單

### 第 1 關：新帝國主義
- **故事摘要**：工業革命帶動對市場與資源的需求。社會達爾文主義、「白種人的負擔」等思想促使歐洲列強向亞洲與非洲擴張，造成亞非多數地區淪為殖民地，而拉丁美洲則受到列強的經濟控制。
- **測驗題目**：
  1. **19 世紀末期，歐洲列強積極在海外進行「新帝國主義」擴張，其最主要的經濟動機是什麼？**
     - [ ] (A) 為了學習當地古老文明與先進技術
     - [x] (C) 尋找更廣大的商品市場與廉價的工業原料
     - [ ] (D) 幫助當地建立現代化的民主政府
  2. **當時列強常以「優勝劣敗，適者生存」的理論來合理化侵略行為，此思想稱為什麼？**
     - [ ] (A) 啟蒙運動
     - [x] (B) 社會達爾文主義
     - [ ] (C) 孤立主義
  3. **新帝國主義時期，歐洲列強對亞洲、非洲與拉丁美洲造成了什麼影響？**
     - [x] (B) 非洲幾乎被瓜分殆盡，亞洲多處淪為殖民地，而拉丁美洲則受到外國的經濟資本控制
     - [ ] (C) 三大洲皆成功抵抗了歐洲勢力的入侵
- **地圖拖拉配對挑戰（新增！）**：
  - **機制**：上方提供 4 個殖民母國選項標籤（英國、法國、荷蘭、美國）。下方為一張**無文字標記的東南亞/南亞地圖**，地圖上有 4 個挖空的放置點。
  - **配對目標（拖放至地圖對應地理位置）**：
    - 印度區域 ⬅️ 英國
    - 印尼區域 ⬅️ 荷蘭
    - 越南區域 ⬅️ 法國
    - 菲律賓區域 ⬅️ 美國

### 第 2 關：戰雲密布
- **故事摘要**：歐洲列強因利益衝突結成兩大同盟：三國同盟（德、奧、義）與三國協約（英、法、俄）。其中巴爾幹半島因民族與宗教衝突頻繁，被稱為「歐洲火藥庫」。
- **分類拖拉挑戰（取代原有的選擇題 1、2 題與配對挑戰）**：
  - **機制**：畫面上方列出 6 個國家選項（德國、英國、法國、俄國、奧匈帝國、義大利）。下方有兩個大分類區塊：「三國協約」與「三國同盟」，每個區塊內各有 3 個空格。學生必須將國家拖拉到對應的陣營中。
  - **配對目標**：
    - 三國同盟 ⬅️ 德國、奧匈帝國、義大利
    - 三國協約 ⬅️ 英國、法國、俄國
- **測驗題目**：
  1. **第一次世界大戰爆發前，哪個地區因為民族組成複雜與宗教衝突不斷，被稱為「歐洲火藥庫」？**
     - [x] (A) 巴爾幹半島
     - [ ] (B) 伊比利半島
     - [ ] (C) 義大利半島
     - [ ] (D) 斯堪地那維亞半島

### 第 3 關：第一槍與戰火蔓延
- **故事摘要**：1914 年塞拉耶佛事件爆發（奧國皇儲遇刺），引爆同盟與協約陣營的對立，演變成世界大戰。大戰初期西線戰場陷入「壕溝戰」的僵持局面。
- **測驗題目**：
  1. **導致第一次世界大戰全面爆發的「導火線」事件是什麼？**
     - [x] (D) 塞拉耶佛事件（奧國皇儲遇刺）
  2. **大戰爆發後，西線戰場陷入長期的防守僵持，主要採用了何種作戰型態？**
     - [x] (B) 壕溝戰
  3. **大戰初期，有一個國家原本屬於「三國同盟」，但在開戰後不久卻轉而加入「協約國」陣營，這是哪個國家？**
     - [x] (A) 義大利

### 第 4 關：戰局轉折與結束
- **故事摘要**：1917 年德國實施「無限制潛艇政策」引發美國參戰；同年俄國爆發共產革命而退出大戰。1918 年 11 月，同盟國陣營瓦解，大戰結束。
- **測驗題目**：
  1. **1917 年，導致美國決議加入協約國參戰的最直接原因是什麼？**
     - [x] (B) 德國實施「無限制潛艇政策」，屢次擊沉美國船隻
  2. **同年（1917 年），協約國陣營中哪個國家因為國內爆發革命而提早退出戰場？**
     - [x] (D) 俄國
  3. **歷時四年多的第一次世界大戰，最後在哪一年因德國投降而宣告結束？**
     - [x] (C) 1918 年
- **挑戰**：排序挑戰（排列歷史事件：塞拉耶佛事件 -> 大戰爆發 -> 壕溝戰僵持 -> 俄國退出與美國參戰 -> 大戰結束）。

### 第 5 關：巴黎和會與國際聯盟
- **故事摘要**：1919 年由美英法主導召開巴黎和會。美國總統威爾遜提出「十四點原則」，強調「民族自決」與成立「國際聯盟」。但美國反而受孤立主義影響未加入國聯。
- **測驗題目**：
  1. **1919 年召開的巴黎和會，用以懲罰戰敗國與重整歐洲秩序，該會議主要由哪三個戰勝國主導？**
     - [x] (C) 美、英、法
  2. **美國總統威爾遜在戰後提出了「十四點原則」，其中包含哪一個重要主張，促成東歐許多新獨立國家誕生？**
     - [x] (A) 民族自決
  3. **戰後各國依據威爾遜的建議成立了哪一個國際組織，以維護世界和平？（儘管美國並未加入）**
     - [x] (C) 國際聯盟
- **挑戰**：名詞與定義配對挑戰（關於凡爾賽和約、民族自決、國際聯盟等）。
